1. 프로그램 소개.

안녕하세요 노블아크(NOBLE ARC) 유튜브 영상 자동화 솔루션 프로그램에 대해서 지금부터 자세히 설명드리겠습니다.먼저 프로그램에 대해서 어떤 프로그램인지 소개해드리겠습니다.

이 부분을 skip하고 다른 부분을 먼저 보고싶은 분은 목차를 클릭하시면 해당 설명 부분으로 이동하실 수 있습니다.

저희 프로그램은 유튜브 영상 제작의 모든 것을 전부 자동화 하는 프로그램입니다. 유튜브 영상 소재를 찾는 것부터 시작해서 최상위 품질의 대본. 그리고 대본에 맞는 이미지 자동생성. 그리고 고품질 음성 자동생성. 그리고 자막 자동생성 및 음성과 완벽한 씽크 맞춤 자동화. 그리고 이 모든 것을 합하여 편집하는 것까지. 클릭 몇번 또는 단 한번의 클릭으로 전자동 시스템으로 완성형 유튜브 영상을 만드는 프로그램입니다.(제목,설명문구,태그 등 메타데이터와 유튜브 업로드까지도 자동화 가능)

수년간 축적된 유튜브 노하우와 인기 영상 제작비법을 프로그램 로직에 그대로 담았습니다. 단순한 자동화시스템이 아닙니다. 실제로 성과를 낼 수 있도록 설계되어 있습니다.

또한 가장 중요한 것 중 한 가지는 유튜브 영상을 자동으로 생성하는 과정에 있어 비용을 최소화하면서도 고품질의 영상이 만들어지는 구조로 되어 있다는 것입니다. 프로그램을 이용하는 구매자의 비용을 고려하지 않고 영상을 자동화하는 타사의 프로그램과는 차별점이 분명히 있습니다.

그리고 또 한가지는 프로그램에 영상 제작에 모든 것을 지원하고 있으면서 프로그램 이용료는 매우 저렴한 가격을 유지하고 있다는 점입니다.

유튜브에 도전하시는데 있어 노블아크(NOBLE ARC) 프로그램은 여러분의 비용을 최소화로 절감시킬 뿐만 아니라. 유튜브 영상 제작에 대한 수고로움을 날려버리며 동시에 여러분의 시간까지도 절약시켜주는 최상의 도구임을 말씀드립니다.

저희 프로그램은 9:16 비율의 짧은 타임 영상 제작 프로그램인 쇼츠 전용 자동화 프로그램이 있고 16:9 비율의 긴타임 영상 제작프로그램인 롱폼 전용 자동화 프로그램이 있습니다. 총 2가지의 프로그램이 있습니다.

5.롱폼 전용프로그램 사용법

지금부터 NOBLE ARC 롱폼 유튜브 자동화 프로그램 사용법에 대해서 자세히 설명드리겠습니다.

저희 NOBLE ARC 롱폼 유튜브 자동화 프로그램은 상당히 직관적이고 단순한 인터페이스를 가지고 있어서 조금만 사용하셔도 금방 익숙해지실 수 있습니다. 그러면서도 많은 기능을 담고 있습니다. 타사 대비 더 많은 기능을 가지고 있는 프로그램입니다. 지금 부터 시작합니다.

VERTEX AI JSON을 활용하면 영상 제작의 모든 것을 비용 없이 만들 수 있습니다. 그 외에는 프로그램 사용자의 선택에 따라 추가 기능을 이용하여(API 비용이 발생하는) 영상제작도 가능합니다.

위의 화면은 설치하신 후 라이선스 인증까지 마치셨다면 나오는 첫 화면입니다.

첫 화면은 홈/생성 탭으로써 여기서 대본에 생성에 대한 것들을 결정하게 됩니다.

첫 화면에서 가장 위에 있는 것이 바로 언어 설정입니다. 저희 NOBLE ARC 유튜브 자동화 프로그램은 총 7개의 언어를 지원하며 계속해서 늘려나갈 계획입니다. 특히 한국어와 영어, 일본어는 유튜브 채널을 운영하는데 있어 중심 축을 할 언어입니다. 가장 기본적으로는 한국어가 선택되어 있습니다. 하지만 영어 유튜브 채널을 운영하고자 하시는분 또는 일본어로 유튜브 채널을 운영하고자 하시는분들은 해당 언어를 선택하시면 됩니다. 기타 다른 언어들도 마찬가지 입니다.

자동화시스템이 각 언어에 맞는 최적화된 대본을 생성합니다. 영어와 일본어 스페인어 등을 모르셔도 상관없이 고품질의 대본이 각 언어에 맞게 인공지능에 의해서 생성되며 이미지, 음성더빙까지 모두 각 언어로 운영 가능하도록 NOBLE ARC 유튜브 자동화 프로그램이 지원합니다.

특히 일본어의 경우 한국보다 조회수 1회당 광고 단가가 1.4배 정도로 높은편이고 겸업금지문화가 있어 유튜브를 운영하는 채널의 개수가 인구수 대비 현저히 적은편입니다. 그런데 성격은 한국인보다 조금 더 여유로워 한번 본 영상을 끝까지 보는 경향도 있습니다. 이것은 유튜브 알고리즘의 중요한 요소인 시청시간에 영향을 미치는 요소이기 때문에 알고리즘에 노출도를 높일 수 있는 유리한 환경입니다. 추가적으로 인구도 한국보다 2배이상 많고 유튜브 이용자도 많기 때문에 조금은 포화상태인 국내보다 일본이나 영어권 유튜브 채널 운영도 노려보시길 추천드려봅니다. 여러분의 더 많은 도전과 성공을 위해 다국어 시스템을 도입했습니다.

우선 테스트 과정을 보여드려야 하니 여기서는 한국어로 보여드리겠습니다.

언어설정 다음으로는 영상의 길이와 카테고리 설정이 있습니다. 어떤 길이와 카테고리를 선택하느냐에 따라 해당 부분에 맞는 최적화된 고품질 대본이 생성되기 때문에 선택은 매우 중요합니다. 롱폼 영상의 경우 보통 10~20분 길이로 하실 것을 추천드립니다.

각 영상 길이에 따라 씬 개수(이미지 개수)가 달라지며 카테고리 선택에 따라 대본의 구성이 달라집니다.

다음으로 키워드 생성과 벤치마킹 탭을 보실 수 있습니다. 키워드 생성이란 단순히 키워드나 주제만 입력하시면 알아서 프로그램이 거기에 맞는 대본을 생성해주는 시스템입니다. 좀더 구체화된 주제를 입력할 수록 본인이 의도하는 영상이 나올 수 있습니다. 예를들어 은, 금 이런식으로 하나의 단어로 입력하시기 보다는 은의 특성, 전세계 금의 역할과 같은 식으로 조금 더 구체화된 주제나 키워드를 넣으시길 권장드립니다.

그 옆에는 벤치마킹 탭이 있는데 해당 벤치마킹 탭은 리서치센터에서(리서치센터 사용법은 목차에서 확인 후 이동하시면 자세한 설명을 보실 수 있습니다.) 발굴한 벤치마킹 영상 대본을 입력하시거나 뉴스를 입력하시면 그것을 인공지능이 자동으로 재편집 및 재창조하여 새로운 고품질 대본을 생성하여 줍니다. 벤치마킹 대본의 장점은 살리면서도 고품질의 대본을 새롭게 만들어줍니다.

그래서 사용자의 의도에 따라 대본 생성 방식을 키워드 생성 또는 벤치마킹 대본 생성으로 선택하셔서 이용하시면 됩니다.

또한 직접입력 탭이 있습니다. 직접 작성한 대본이나 다른 곳에서 생성한 대본이 있다면 이곳에 붙여넣기 하여 AI 대본 생성 없이 바로 영상을 만들 수 있습니다.

직접입력 탭을 선택하시면 대본을 직접 붙여넣을 수 있는 텍스트 입력창이 나타납니다. 대본을 입력하시면 하단에 글자수, 예상 씬 수, 권장 영상 길이가 자동으로 계산되어 표시됩니다. 입력 완료 후 딸깍 또는 단계별 생성 버튼을 누르면 AI 대본 생성 단계는 건너뛰고 이미지 생성·음성 더빙·자막·렌더링 과정이 자동으로 진행됩니다.

단계별 생성의 경우에는 대본이 생성되면 대본 확인을 하는 단계를 거치고 이미지가 생성되면 이미지 확인 과정을 거치고 중간에 이미지를 영상으로 전환을 하거나. 음성 선택을 하거나 등등 몇가지 필요한 사항을 선택하면서 단계별로 자동화 진행시키는 버튼입니다. 단계별 생성도 선택만 하면 대본과 이미지 음성더빙, 자막생성 등 모든 것이 자동화로 이루어지며 랜더링 시스템으로 영상화로 마지막 편집까지 자동으로 이루어집니다.

첫 화면에서 가장 위에 있는 것이 바로 언어 설정입니다. 저희 NOBLE ARC 유튜브 자동화 프로그램은 총 7개 의 언어를 지원하며 계속해서 늘려나갈 계획입니다. 특하 한국어와 영어, 일본어는 유튜브 채널을 운영하는데 있어 중심 축을 할 언어입니다. 가장 기본적으로는 한국어가 선택되어 있습니다. 하지만 영어 유튜브 채널을 운 영하고자 하시는분 또는 일본어로 유튜브 채널을 운영하고자 하시는분들은 해당 언어를 선택하시면 됩니다. 기타 다른 언어들도 마찬가지 입니다.

자동화시스템이 각 언어에 맞는 최적화된 대본을 생성합니다. 영어와 일본어 스페인어 등을 모르셔도 상관없 이 고품질의 대본이 각 언어에 맞게 인공지능에 의해서 생성되며 이미지, 음성더빙까지 모두 각 언어로 운영 가 능하도록 NOBLE ARC 유튜브 자동화 프로그램이 지원합니다.

특히 일본어의 경우 한국보다 조회수 1회당 광고 단가가 1.4배 정도로 높은편이고 겸업금지문화가 있어 유튜 브를 운영하는 채널의 개수가 인구수 대비 현저히 적은편입니다. 그런데 성격은 한국인보다 조금 더 여유로워 한번 본 영상을 끝까지 보는 경향도 있습니다. 이것은 유튜브 알고리즘의 중요한 요소인 시청시간에 영향을 미 치는 요소이기 때문에 알고리즘에 노출도를 높일 수 있는 유리한 환경입니다. 추가적으로 인구도 한국보다 2배 이상 많고 유튜브 이용자도 많기 때문에 조금은 포화상태인 국내보다 일본이나 영어권 유튜브 채널 운영도 노 려보시길 추천드려봅니다. 여러분의 더 많은 도전과 성공을 위해 다국어 시스템을 도입했습니다. 우선 테스트 과정을 보여드려야 하니 여기서는 한국어로 보여드리겠습니다.

쇼츠 전용 자동화 프로그램은 5가지의 탭을 지원합니다. 키워드생성, 벤치마크대본, 원본가공, 썰/야담, 직접입력

모두 쇼츠 대본 구성에 맞는 최적화된 고품질 대본을 만들어줍니다.

딸깍 버튼은 키워드나 벤치마킹 대본을 입력하고 해당 딸깍 버튼을 누르면 대본 생성과 이미지 생성 그리고 음 성더빙과 자막생성까지의 과정이 모두 한번에 자동화되어 이루어지며. 이 모든것을 하나로 합쳐 영상으로 만 들어내는 과정까지!! 모든 전 과정을 자동화하는 기능입니다.

단계별 생성의 경우에는 대본이 생성되면 대본 확인을 하는 단계를 거치고 이미지가 생성되면 이미지 확인 과 정을 거치고 중간에 이미지를 영상으로 전환을 하거나. 음성 선택을 하거나 등등 몇가지 필요한 사항을 선택하 면서 단계별로 자동화 진행시키는 버튼입니다. 단계별 생성도 선택만 하면 대본과 이미지 음성더빙, 자막생성 등 모든 것이 자동화로 이루어지며 랜더링 시스템으로 영상화로 마지막 편집까지 자동으로 이루어집니다.

추가적으로 딸깍 버튼의 경우 현재 프로그램 내에 세부설정, 이미지스타일, 음성 선택 등 이미 선택되어 있는 것을 기준으로 영상을 전자동화하여 생성하기 때문에 딸깍 버튼으로 계속해서 영상을 같은 스타일로 만드실 분들은 미리 각 설정 사항들을 설정을 해놓으시고 딸깍 버튼으로 영상을 생성하시기 바랍니다. 각종 세부설정 을 한번 지정해놓으면 그 다음부터는 주제 및 벤치마킹 대본을 넣고 딸깍 버튼을 누르시는 것만으로도 연속해 서 계속 한번의 버튼 클릭으로 영상 생성이 자동화 됩니다.

벤치마킹 탭이 있는데 해당 벤치마킹 탭은 리서치센터에서(리서치센터 사용법은 목차에서 확인 후 이동하시면 자세한 설명을 보실 수 있습니다.) 발굴한 벤치마킹 영상 대본을 입력하여 대본을 생성하는 곳입니다. 인공지능이 자동으로 재편집 및 재창조하여 새로운 고품질 대본을 생성하여 줍니다. 벤치마킹 대본의 장점은 살리면서도 고품질의 대본을 새롭게 만들어줍니다.

원본가공의 경우에는 블로그 글, 나무위키 글, 뉴스 글 등을 복사 붙여넣기 하시면 인공지능이 자동으로 재편집 및 재창조하여 쇼츠에 맞는 새로운 고품질 대본을 생성하여 줍니다.

썰/야담의 경우에는 인터넷에 떠도는 썰이나 야담을 넣으시거나 인공지능에게 썰을 만들어달라고 해서 복사 붙여넣기 하시면 쇼츠에 맞는 최적화된 대본이 만들어지게 됩니다. 또는 썰 주제나 키워드를 입력하시면 알아서 대본이 생성됩니다.

썰유형과 말투를 설정하실 수 있습니다.

딸깍 버튼과 단계별 생성 버튼을 클릭하면 위의 화면과 같이 길이 선택이 있습니다 45초~60초 길이 선택을 추천드립니다. 길이를 선택하시면 대본이 생성되게 됩니다.

마지막으로 직접입력 탭입니다. 직접 작성한 대본이나 다른 곳에서 생성한 대본이 있다면 이곳에 붙여넣기 하여 AI 대본 생성 없이 바로 쇼츠 영상을 만들 수 있습니다.

직접입력 탭을 선택하시면 대본을 직접 붙여넣을 수 있는 텍스트 입력창이 나타납니다. 사용법 안내처럼 작성한 대본을 붙여넣으면 문장 단위로 씬이 자동 분리되어 영상으로 만들어집니다. 마침표(.), 물음표(?), 느낌표(!) 기준으로 씬이 분리되며 선택한 영상 길이(씬 수)를 참고해 1~2문장씩 묶어줍니다. 대본 입력 후 하단에 글자수, 예상 씬 수, 권장 영상 길이가 자동 계산되어 표시됩니다. 딸깍 또는 단계별 생성 버튼을 누르면 AI 대본 생성 단계는 건너뛰고 이미지 생성·음성 더빙·자막·렌더링 과정이 자동으로 진행됩니다.

Google 검색 기반 대본 생성

키워드 생성, 벤치마크 대본, 원본가공, 썰/야담 탭 사용 시 하단에 Google 검색 기반 대본 생성 체크박스가 표시됩니다. 이 기능은 대본 생성 과정에서 Google 검색을 실시간으로 참조하여 최신 정보와 사실을 대본에 반영해주는 신규 기능입니다.

체크하시면 대본 생성 시 AI가 Google 검색 결과를 추가로 참조하여 최신 이슈, 수치, 사건 등을 반영한 대본을 만들어줍니다. 과거 학습 데이터는 그대로 유지하면서 최신성과 정확성이 동시에 강화됩니다.

최신 정보가 중요한 주제(시사, 뉴스, 트렌드 등)에서 특히 효과적입니다. 단, 검색 과정이 추가되어 생성 시간이 다소 늘어날 수 있습니다. 최신 정보가 필요하지 않은 주제라면 체크를 해제하고 사용하셔도 됩니다.

다음은 대본검토 탭입니다. 단계별 생성으로 대본을 생성하면 여기서 대본 검토를 하실 수 있습니다. 각 씬별로 대본이 분리되어 보이며 위의 전체 복사를 통해 대본을 복사하실수도 있습니다.

대본 검토가 끝났다면 대본 재생성 버튼을 누르시거나(대본이 마음에 안드신경우) 다음 단계인 세부설정으로 이동하기 버튼을 누르시면 됩니다. 대본 재생성 버튼을 누르실경우 처음 홈/생성 탭으로 이동하여 단계별 생성 버튼을 한번 더 누르실 수 있습니다. 단계별 생성 버튼을 한번 더 누르시면 대본이 재생성 되며 다시 대본 검토 탭으로 이동하여 다시 검토하실 수 있습니다

다음으로 세부설정 탭입니다. 여기서 선택하신 사항은 딸깍 버튼에도 모두 적용됩니다. 가장 처음에는 성우를 선택하는 겁니다. 저희 NOBLE ARC 유튜브 자동화 프로그램은 총 4가지의 TTS 음성더빙 시스템을 지원합니 다. 가장 기본은 Gemini TTS 입니다. 발급받으신 무료 구글 크레딧에서 크레딧이 소모되면서 음성더빙이 생 성 되기 때문에 무료입니다. 품질도 상당히 고품질입니다. 각 음성별로 미리듣기가 지원되고 있어 미리 들어보 시고 음성을 선택하시면 됩니다. 무료이면서 고품질이지만 다른 TTS 대비 생성 속도가 조금 느립니다.

하지만 자동화 프로그램 특성상 자동화하고 다른일을 하시면서 영상을 만드실 수 있기 때문에 무료로 Gemini TTS로 생성하시는 것을 권장드립니다. 좀 더 고품질의 TTS 음성 더빙을 원하실경우 MINIMAX TTS 도는 일레븐랩스 TTS를 이용하시면 됩니다.

캐릭터 참조 이미지 업로드 (이미지 직접 사용)

텍스트로 캐릭터를 설명하는 방식 외에, 직접 보유하고 있는 캐릭터 이미지를 업로드하여 해당 캐릭터를 모든 씬에 반영시킬 수 있습니다. 세부설정의 캐릭터 탭에서 이미지 생성 모델 아래 캐릭터 참조 이미지(선택) 항목을 찾으실 수 있습니다.

SNS 콘텐츠 변환 — 영상 대본으로 SNS 글 자동 발행

영상 완성 후 렌더링 탭에는 SNS 콘텐츠 변환 기능이 추가되어 있습니다. 완성된 영상의 대본을 바탕으로 각 SNS에 최적화된 글을 자동으로 생성하여 바로 활용하실 수 있습니다.

생성된 글은 해시태그와 함께 출력되며 영상 이미지도 함께 표시됩니다. 전체 복사 버튼으로 내용을 바로 복사하여 각 SNS에 붙여넣기 하시면 됩니다. 쇼츠 영상 업로드와 SNS 홍보를 동시에 진행할 수 있어 채널 운영 효율이 크게 높아집니다.

쿠팡파트너스 쇼츠 사용법

쿠팡파트너스 쇼츠는 쿠팡 상품을 키워드로 검색하여 해당 상품을 소개하는 쇼츠 영상을 자동으로 제작하고, 유튜브에 자동 업로드까지 연결해주는 기능입니다. 영상 설명란과 댓글에 쿠팡 파트너스 구매 링크가 자동으로 포함되어 수익화에 활용하실 수 있습니다.

⚠️ 이 기능을 사용하려면 쿠팡파트너스 API 키(Access Key · Secret Key)가 필요합니다. 쿠팡파트너스 API 키는 쿠팡파트너스 센터에서 발급받으실 수 있으며, 파트너스 활동을 통해 15만 원 이상의 실적을 쌓아야만 API 키 발급 신청이 가능합니다. 아직 실적이 없으신 경우 먼저 일반 파트너스 활동으로 실적을 쌓으신 후 API 키를 발급받으시기 바랍니다.

API 키를 발급받으셨다면 프로그램의 API 키 관리 탭에서 쿠팡파트너스 API 섹션을 찾아 Access Key와 Secret Key를 각각 입력하고 저장 버튼을 눌러주세요. 연결 테스트 버튼으로 키가 정상적으로 인식되는지 확인하실 수 있습니다.

API 키 등록이 완료되면 왼쪽 메뉴에서 쿠팡파트너스 쇼츠 탭으로 이동합니다. 탭에는 단일 실행과 자동화 두 가지 모드가 있습니다. 단일 실행은 한 번에 상품 하나를 골라 영상 한 개를 제작하는 방식이고, 자동화는 여러 키워드를 등록해두면 순서대로 영상을 자동으로 계속 제작하는 방식입니다.

단일 실행 방법입니다. 상품 검색 영역에서 원하시는 상품 키워드를 입력하고 개수를 설정한 뒤 검색 버튼을 누르면 쿠팡에서 관련 상품이 목록으로 표시됩니다. 목록에서 영상으로 만들 상품을 클릭하여 선택하세요.

상품을 선택하면 하단에 선택된 상품 정보와 파트너스 URL이 자동으로 생성됩니다. YouTube 자동 업로드, 설명란에 쿠팡 링크 포함, 댓글에 구매 링크 자동 등록 옵션을 원하시는 대로 체크하신 후 영상 생성 시작 버튼을 눌러주세요. 원하시는 배경음악이 있다면 BGM 설정에서 업로드해 적용하실 수 있습니다.

자동화 방법입니다. 자동화 탭에서는 키워드 목록란에 한 줄에 하나씩 키워드를 입력합니다. 입력된 키워드 순서대로 자동으로 상품을 검색하고 영상을 제작합니다. 상품 선택은 AI가 자동으로 상위 5개 중 최적 상품을 선택하며, 영상 간 대기 시간은 한 영상이 완성된 후 다음 영상 제작을 시작하기까지의 대기 시간으로 30분 이상 설정하시길 권장합니다.

키워드 입력 후 반드시 저장 버튼을 눌러주세요. 팝업창이 뜨면 확인을 눌러 저장을 완료하신 다음, 자동화 시작 버튼을 누르시면 설정된 키워드 순서대로 영상이 자동으로 만들어집니다.

여기까지 NOBLE ARC 유튜브 자동화 쇼츠 전용 프로그램에 대한 사용법 설명을 마치겠습니다.

7.리서치센터 사용법

자 이제부터 리서치센터에 대해서 자세히 알려드리겠습니다. 저희 NOBLE ARC 프로그램은 리서치센터를 통해 좋은 소재를 발굴할 수 있는 시스템을 지원해드리고 있습니다. 유튜브에서 엄청난 조회수를 기록한 떡상 인기 영상을 검색해서 찾아낼 수 있고 그 영상을 벤치마킹 할 수 있고 경쟁자 채널분석을 통해 유명 유튜버들의 유튜브 채널 운영 형태를 분석해낼 수도 있습니다. 또한 최근 트렌드 검색을 통해 뜨고있는 검색어 키워드를 쉽게 찾아 낼 수 있으며 해당 키워드의 뉴스 정보도 쉽게 검색해서 소재를 발굴하실 수 있도록 지원해드리고 있습니다. 리서치센터를 이용하시기 위해서는 유튜브 API키를 반드시 발급 받으셔야 합니다. 아직 발급을 받지 않으셨다면 4-6 목차에서 유튜브 API키 발급 방법 설명을 보시면서 먼저 발급받고 오시기를 권해드립니다.

자 그러면 각 기능별로 자세히 설명드리겟습니다.

롱폼 프로그램에서의 리서치센터, 쇼츠프로그램에서의 리서치센터 모두 사용법이 동일합니다.

7-1떡상영상 검색법

먼저 떡상영상(인기영상 또는 뜨고 있는 영상)검색법에 대해서 설명드리겠습니다.

NOBLE ARC 프로그램 리서치센터로 들어가셔서 떡상검색 탭을 클릭합니다.

밑에 보시면 유튜브 떡상 검색란이 있습니다. 여기에 원하시는 키워드를 입력하시면 됩니다. 경제라고 입력하시거나 최근 유가 상승 등 포괄적이거나 좀더 구체적인 키워드를 입력해 가시면서 떡상 영상을 검색하실 수 있습니다. 기본적으로는 기간은 1주일로 선택되어 있고 길이는 4~20분으로 설정되어 있습니다. 키워드 입력 후 오른쪽에 있는 데이터 수집 시작 버튼을 클릭합니다.

원하시는 경우 기간을 조정해서 더 짧은기간 또는 더 긴 기간에서의 떡상 영상을 검색하실 수 있습니다.

떡상 지수가 높을수록 더 유튜브 알고리즘의 선택을 받은 것이며 시청자들에게 더욱 주목을 받은 상태라고 보실 수 있습니다. 그런데 구독자 500명 이상 조회수 10000이상의 영상부터 의미가 있으니 구분해서 보시기 바랍니다.

검색 결과에서 마음에 드는 영상을 발견하셨다면 해당 영상의 대본을 바로 복사하실 수 있습니다. 영상 하단에 있는 대본복사 버튼을 클릭하시면 됩니다. 버튼을 바로 누르시면 해당 영상의 대본 전체가 복사됩니다.

대본복사 버튼을 클릭하시면 복사 완료라는 메시지가 표시됩니다. 이렇게 복사가 완료된 것을 확인하실 수 있습니다.

복사가 되었다면 NOBLE ARC 프로그램 홈/생성 탭에서 벤치마킹 탭을 선택하신 후 참고 텍스트 란에 붙여넣기 하시면 됩니다. 그러면 새롭게 벤치마킹한 새로운 대본이 생성되게 됩니다. 언어를 일본어나 영어로 선택하시면 일본어나 영어로 새로운 대본이 생성 되게 됩니다.

떡상 영상 검색기만으로도 좋은 소재를 얼마든지 발굴해낼 수 있습니다.

7-2경쟁자 채널 분석법

다음으로 유튜브 경쟁자 채널(또는 인기채널)을 분석하는 방법을 안내해드리겠습니다. 해당 분석 탭을 이용하실 경우 경쟁자 채널 또는 인기 채널의 업로드 패턴 그리고 인기영상과 조회수가 저조한 영상 그리고 최적의 업로드 시간과 수익성 등등 다양한 정보를 분석해보실 수 있습니다.

먼저 리서치센터 탭으로 들어가셔서 경쟁자 분석 버튼을 클릭해줍니다. 그러면 위와 같은 화면이 나오는데 채널 핸들 또는 URL 입력 화면이 나옵니다.

경쟁자 채널 또는 인기채널의 채널 핸들을 알아내는 방법은 해당 채널로 들어가셔서 먼저 더보기를 클릭하시고 설명과 추가정보가 나오는 화면에서 @채널핸들이름을 그대로 드래그해서 복사하세요

그것을 프로그램 채널핸들 입력란에 붙여넣기 하고 분석 버튼을 누르시면 됩니다. 그럼 위 화면처럼 다양한 분석 탭이 등장합니다. 해당 정보를 참고하시면 유튜브 채널을 운영하시는데 더욱 도움이 될 수 있습니다.

7-3최근 트렌드 검색어 및 이슈 뉴스 검색법

다음으로 최근 트랜드 검색어 및 이슈 뉴스 검색법입니다. 해당 부분을 이용하면 떡상검색 못지 않은 다양한 소재를 발굴해 낼 수 있습니다.

뉴스트렌드로 들어가시면 두가지 버튼이 나오는데 하나는 실시간으로 인기 검색어 트렌드를 알아낼 수 있는 탭이고 또 하나는 각 키워드별로 최신 뉴스들을 검색하는 탭입니다. 실시간 트렌드에서 인기 검색어가 나오면 그 인기검색어를 눌렀을때 해당 검색어에 해당하는 최신 뉴스들이 뜨는 기능도 있습니다. 실시간 트렌드에서 뜨고있는 검색어를 입력하셔서 어떤 이유로 이 검색어가 뜨고 있는지 뉴스들을 통해 알 수 있고 해당 뉴스들을 복사해서 NOBLE ARC 프로그램 벤치마킹에 복사 붙여넣기 하면 최신 트렌드로 유튜브 영상을 제작할 수 있습니다.

실시간 트렌드를 눌렀을때 뜨는 최신 실시간 인기 검색어 트렌드입니다. 해당 검색어들 중 마음에 드시는 것을 클릭해보세요

보시다시피 인기 트렌드 검색어를 클릭하면 관련 최신 뉴스 기사들이 나오는 것을 보실 수 있습니다. 해당 뉴스를 클릭하면 새로운 창이 뜨면서 해당 뉴스를 보실 수 있습니다.

이제 뉴스 내용을 드래그 하셔서 NOBLE ARC 프로그램의 홈/생성 탭에서 벤치마킹을 클릭하시고 뉴스 내용을 붙여넣기 해보세요

9.쇼츠 영상 대량생산탭 운영방법

대량생산 탭은 AI가 자동으로 영상 소재를 생성하고, 대기열에 등록된 순서대로 영상을 연속 제작하는 완전 자동화 도구입니다. 쇼츠 전용 기능이며, 완성 후 YouTube 자동 업로드까지 연결할 수 있습니다. 다만 YouTube API를 통한 자동 업로드는 1일 6회까지만 가능하므로 참고하시기 바랍니다.

영상을 하나하나 만드는 번거로움 없이 한 번의 설정으로 수십 개의 영상을 자동으로 생산할 수 있습니다.

★ 대량 생산 전 반드시 확인하세요 ★
대량 생산을 시작하기 전에 언어 설정, 세부 설정, 이미지 스타일을 원하시는 스타일로 미리 선택해 놓으시기 바랍니다. 선택된 설정을 기반으로 영상이 연속 자동 생성되므로, 사전 설정이 매우 중요합니다.

① AI 소재 생성

사이드바에서 ⑥ 대량 생산 탭으로 이동합니다. 상단의 AI 소재 생성 카드에서 아래 항목을 설정해 주세요.

  • 키워드 — 만들고 싶은 영상 주제 키워드를 입력합니다. (예: 우주, 심리학, 재테크 등)
  • 생성 수량 — 한 번에 생성할 소재 수를 선택합니다. (10 / 20 / 30 / 40 / 50개)
  • 카테고리 — 영상 채널 카테고리를 선택합니다.
  • 분위기 — 영상의 전반적인 분위기와 톤을 선택합니다.

설정 완료 후 ✨ AI 소재 생성 버튼을 클릭하면, AI가 키워드를 바탕으로 흥미로운 쇼츠 주제를 자동으로 생성합니다. 생성된 소재 목록이 체크박스 형태로 표시되며, 전체 선택 또는 원하는 항목만 개별 선택이 가능합니다. 마음에 들지 않는 소재는 체크를 해제하고, 마음에 드는 항목만 선택하세요.

② 대기열 추가 및 대량 생산 시작

원하는 소재를 선택한 후 📋 선택 항목 대기열에 추가 버튼을 클릭하면 하단 대기열 현황에 항목이 등록됩니다.

대기열 등록 사항을 확인하신 후, 다음 설정을 조정합니다.

  • 영상 간 대기 시간 — 영상 1개 완성 후 다음 영상 시작까지의 대기 시간을 설정합니다.
  • 실패 시 재시도 — 오류 발생 시 자동으로 재시도할 횟수를 설정합니다.
  • YouTube 자동 업로드 — 완성된 영상을 자동으로 YouTube에 업로드하려면 체크합니다. (1일 최대 6회)

모든 설정이 완료되면 🚀 대량 생산 시작 버튼을 누릅니다. 이후 등록된 소재 수만큼 영상이 자동으로 연속 제작됩니다. 대량 생산이 진행되는 동안 다른 작업을 하셔도 무방하며, 완료 후 결과 카드의 📂 OUTPUT 폴더 열기 버튼을 클릭하면 완성된 영상이 저장된 폴더가 바로 열립니다.

10.이미지 영상으로 전환 후 씬 편집기에서 영상 등록 방법

먼저 프로그램에서 생성된 이미지를 확인합니다. 이미지를 영상으로 전환할 때는 https://docs.google.com/videos (Google Vids)를 이용하시면 무료로 이미지를 영상으로 전환하실 수 있습니다. 구글 계정만 있으면 별도의 구독 없이 Google Veo 3.1 AI 기술을 활용해 이미지를 자연스럽게 움직이는 영상으로 변환할 수 있습니다. 아래 순서대로 따라하시면 됩니다.

씬 편집기 화면 최상단에 있는 이미지 폴더 열기 버튼을 클릭합니다. 이미지 생성이 완료된 시점에서 진행하세요.

클릭하면 프로그램이 생성한 이미지들이 씬 번호별로 표시됩니다. 이 목록에서 영상으로 전환할 이미지를 확인하세요. 모든 이미지를 전환할 필요는 없으며, 초반 3~5개 씬만 영상으로 변환해도 영상의 몰입도를 크게 높일 수 있습니다.

브라우저에서 https://docs.google.com/videos 로 접속합니다. Google Vids 메인 화면에서 새 동영상 시작 버튼을 클릭합니다.

팝업이 나타나면 Veo 3.1 신규를 클릭합니다. 제작하는 영상 유형에 따라 쇼츠는 세로 모드, 롱폼은 가로 모드를 선택하세요.

AI 동영상 클립 패널이 열리면 화면 가장 오른쪽 상단의 드롭다운 버튼을 클릭한 후 이미지에 애니메이션 효과 적용 (이미지를 동영상으로 변환)을 선택합니다.

이미지에 애니메이션 효과 적용 모드로 전환되면 이미지 추가 버튼이 나타납니다. 앞서 이미지 폴더에서 확인한 이미지를 드래그하거나 버튼을 클릭해 업로드합니다.

이미지가 추가되면 텍스트 입력란에 씬 편집기에 있는 해당 씬의 영어 프롬프트를 복사하여 붙여넣기 합니다. 입력 후 하단의 생성 버튼을 클릭합니다.

잠시 후 AI 영상 생성이 완료됩니다. 생성된 영상을 미리보기로 확인하신 후 삽입 버튼을 클릭합니다.

왼쪽 최상단의 파일을 클릭합니다. 그리고 나서 바로 아래 다운로드 버튼을 클릭하세요. 그러면 오른쪽에 mp4 동영상이라고 보입니다. 그걸 클릭하여 다운로드를 합니다.

큰 화면에서 마우스 우클릭을 누른 다음 삭제 버튼을 클릭해주세요.

다음 이미지도 계속 영상으로 전환하려면 입력했던 이미지와 프롬프트 우측의 삭제(X) 버튼을 눌러 초기화합니다.

초기화된 상태에서 다시 새 이미지를 추가하여 계속해서 영상을 제작할 수 있습니다. 전환이 필요한 씬 수만큼 위 과정을 반복하세요.

다운로드한 영상을 프로그램에 등록합니다. 씬 편집기에서 해당 씬의 이미지/영상 버튼을 클릭한 후, 방금 다운로드한 영상 파일을 선택하여 업로드합니다.

업로드가 완료되면 씬 우측 하단에 ▶동영상 표시가 나타납니다. 이것으로 이미지가 영상으로 성공적으로 등록된 것입니다. 렌더링 시 해당 씬은 영상으로 반영되어 최종 결과물에 출력됩니다.

11. AI 에이전트 탭 사용법

AI 에이전트는 NOBLE ARC 프로그램이 사람의 개입 없이 유튜브 채널을 스스로 운영할 수 있도록 설계된 완전 자동화 기능입니다. 단순 반복 작업을 넘어, 소재 발굴부터 완성된 영상의 업로드까지 전 과정을 AI가 주도적으로 처리합니다.

AI 에이전트가 자동으로 처리하는 주요 작업은 다음과 같습니다.

  • 소재 발굴 — 리서치센터의 떡상검색기를 통해 트렌드 영상 소재를 자동으로 탐색합니다.
  • 대본 생성 — 발굴된 소재를 바탕으로 고품질 영상 대본을 자동으로 작성합니다.
  • TTS / BGM / 메타데이터 설정 — 음성 합성, 배경음악 선택, 제목·설명·태그 등 메타데이터까지 자동으로 구성합니다.
  • 이미지 생성 및 영상 제작 — 대본에 맞는 이미지를 AI로 생성하고 완성형 영상을 자동으로 제작합니다.
  • 유튜브 자동 업로드 — 완성된 영상을 설정된 일정에 따라 유튜브 채널에 자동으로 업로드합니다.
  • 댓글 자동 관리 — 업로드된 영상에 달리는 댓글을 설정된 말투와 방식으로 자동으로 관리합니다.
  • 이메일 성과 보고 — 채널 운영 현황과 성과를 정기적으로 이메일로 보고합니다.

초기 설정만 완료하면, 이후에는 AI 에이전트가 유튜브 채널을 자율적으로 운영합니다. 타사 솔루션과의 가장 큰 차별점은 소재 발굴, 제작, 업로드, 채널 관리까지 하나의 플랫폼 안에서 완전히 통합되어 있다는 점입니다.

AI 에이전트는 두 가지 실행 방식을 제공합니다. 용도에 맞게 선택하여 사용하시면 됩니다.

  • 즉시 실행 — 버튼을 클릭하는 즉시 에이전트가 1회 실행된 후 자동으로 종료됩니다. 에이전트 동작을 테스트하거나 일회성으로 실행하고 싶을 때 사용하시면 됩니다.
  • 자동 스케줄 — 에이전트가 설정된 주기에 따라 지속적으로 반복 실행되며, 채널을 꾸준히 운영합니다. 채널을 완전 자동으로 운영하고자 할 때 사용하는 방식입니다.

YouTube API 할당량과 관련하여 중요한 사항을 안내드립니다. 구글에서 제공하는 YouTube API는 하루 사용량에 제한이 있으며, 1개의 구글 계정으로는 5개 이하의 채널 운영을 권장합니다. 또한 채널당 하루 업로드 횟수는 5회 이내로 설정하시는 것이 안정적입니다. 이 기준을 초과하면 API 할당량이 소진되어 업로드가 일시 중단될 수 있으니 참고하시기 바랍니다.

Kling AI를 이용한 초반 씬 영상 변환 기능은 선택 사항입니다. 이 기능을 사용하시려면 목차 4-8번의 Kling AI API 키 발급 안내를 먼저 참고하시기 바랍니다. 초반 씬 변환 개수를 0으로 설정하면 카메라 효과만 적용되며, Kling AI를 사용하지 않습니다.

댓글 자동 관리 기능은 활성/비활성 중에서 선택할 수 있으며, 댓글에 답변할 말투 스타일도 함께 설정할 수 있습니다. 활성으로 설정하면 AI가 댓글을 분석하여 적절한 답변을 자동으로 작성합니다.

AI 에이전트가 운영할 유튜브 채널을 등록하는 섹션입니다. 채널 추가 버튼을 클릭하면 채널 정보를 입력할 수 있는 폼이 열립니다.

채널 추가 폼에서 입력해야 하는 세부 항목들을 설명드립니다.

  • 채널 이름 — 관리를 위한 구분용 이름입니다. 실제 유튜브 채널명과 동일하게 입력하시면 관리가 편리합니다.
  • 업로드 공개 설정 — 영상 업로드 시 공개, 일부 공개, 비공개 중에서 선택합니다.
  • Client ID / Client Secret — 유튜브 자동 업로드에 필요한 인증 정보입니다. 발급 방법은 목차 8번(유튜브 자동업로드 설정 방법)을 참고하시기 바랍니다.
  • Refresh Token — 인증 후 자동으로 입력되는 값입니다. 임의로 수정하지 마시기 바랍니다.
  • 채널 컨셉 — 해당 채널에서 다룰 주제나 콘텐츠 방향을 입력합니다. AI가 이 정보를 바탕으로 소재를 발굴하고 대본을 작성하므로, 가능한 한 구체적으로 작성하실수록 품질이 높아집니다.
  • 하루 영상 수 — 하루에 업로드할 영상의 개수를 설정합니다. YouTube API 할당량을 고려하여 1개를 권장합니다.
  • 에이전트 활성 — 해당 채널에 대한 AI 에이전트 기능을 켜고 끄는 스위치입니다.
  • 채널 언어 — 영상 대본과 메타데이터가 생성될 언어를 선택합니다. 현재 한국어를 포함한 7개 언어를 지원합니다.

AI 에이전트가 영상을 제작할 때 사용할 이미지 스타일을 설정합니다. AI 자동 결정 방식은 채널 컨셉과 대본 내용에 맞는 스타일을 AI가 자동으로 판단하여 적용하며, 스타일 고정 방식은 특정 스타일을 고정하여 채널의 일관된 시각적 아이덴티티를 유지합니다. 스타일 목록 중 '프리미엄 2D'는 고품질 스틱맨 스타일로, 깔끔하고 가독성이 높은 영상 제작에 적합합니다.

채널 설정 화면에서 채널별로 YouTube API 키를 개별적으로 입력할 수 있습니다. 채널마다 서로 다른 API 키를 사용해야 하는 경우에 유용한 기능입니다.

이메일 성과 보고 설정 시 주의사항이 있습니다. 발신 이메일 주소와 수신 이메일 주소에 동일한 Gmail 주소를 입력하시면 됩니다. 비밀번호란에는 Gmail 로그인 비밀번호가 아닌 Gmail 앱 비밀번호(16자리)를 입력해야 합니다.

Gmail 앱 비밀번호는 Google 계정의 2단계 인증이 활성화된 상태에서만 발급받을 수 있습니다. 브라우저에서 myaccount.google.com/security 에 접속한 뒤 '2단계 인증' 항목을 클릭합니다. 2단계 인증 설정 화면 하단으로 스크롤하면 '앱 비밀번호' 항목이 나타납니다. 또는 myaccount.google.com/apppasswords 주소로 바로 접속하셔도 됩니다.

앱 비밀번호 페이지로 이동하면 앱 이름 입력란이 표시됩니다. 이름은 'NOBLE ARC' 또는 본인이 알아볼 수 있는 이름을 자유롭게 입력하시면 됩니다. 이름 입력 후 '만들기' 버튼을 클릭하면 16자리 앱 비밀번호가 생성됩니다. 생성된 16자리 비밀번호를 복사하여 프로그램의 이메일 성과 보고 비밀번호란에 붙여넣기 하시면 설정이 완료됩니다.

초기 설정을 모두 완료하셨다면 이제 AI 에이전트가 여러분의 유튜브 채널을 자동으로 운영합니다. 처음에는 즉시 실행으로 동작을 확인하신 후, 자동 스케줄로 전환하여 완전 자동 운영 체계를 구축하시기 바랍니다. 설정 관련 궁금한 사항은 언제든지 고객 지원을 통해 문의해 주세요.

12. 기타 안내사항

12-1 기기초기화

NOBLE ARC 프로그램은 1대의 PC에서만 이용 가능합니다. 개인적인 사유로 프로그램 사용 기기를 변경하고자 하실 경우, API 키 관리 탭 가장 하단에 있는 기기 초기화 버튼을 눌러주세요.

기기 초기화 버튼을 클릭하면 확인 창이 나타납니다. 확인을 누르면 현재 기기의 등록이 즉시 해제되며 프로그램이 자동으로 종료됩니다.

초기화 완료 후 다른 컴퓨터에서 라이선스 키를 입력하여 접속하시면 새 기기로 등록이 이루어지며, 이후 해당 기기에서 정상적으로 이용하실 수 있습니다.
